Moves a shot sprite.

Moves a shot sprite, checking the collisions with the borders and other sprites. Updates its position. Also checks if it hits the other player.

Parameters
sppointer to the shot sprite.
ppointer to the player to hit sprite.
h_reswidth of the buffer.
v_resheight of the buffer.
video_mempointer to a buffer.
Returns
1-4 if the sprite collides with a border. 5 if it collides with a sprite other that the player's. 6 if it collides with the player. 0 if there are no collisions.
